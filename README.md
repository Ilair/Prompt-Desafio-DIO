# Prompt-Desafio-DIO
Atue como um especialista em educação financeira digital e experiência do usuário.

Crie a proposta de um Educador Financeiro Inteligente voltado para jovens adultos, trabalhadores autônomos e pessoas que desejam organizar melhor suas finanças pessoais.

O sistema deve ajudar o usuário a:

- controlar receitas e despesas;
- organizar o orçamento mensal;
- criar metas financeiras;
- desenvolver uma reserva de emergência;
- aprender conceitos básicos de educação financeira;
- receber recomendações personalizadas para melhorar sua saúde financeira.

As principais funcionalidades devem incluir:

- painel financeiro com receitas, despesas e saldo;
- cadastro de gastos por categorias;
- planejamento de metas financeiras;
- geração de gráficos de evolução financeira;
- alertas de gastos excessivos;
- recomendações inteligentes baseadas no perfil do usuário;
- simulador de economia para atingir objetivos financeiros;
- dicas educativas sobre investimentos, orçamento e consumo consciente.

A Inteligência Artificial deverá adaptar todas as recomendações utilizando informações como:

- renda mensal;
- despesas fixas;
- despesas variáveis;
- idade;
- perfil financeiro;
- objetivos do usuário.

A comunicação deve ser clara, amigável, motivadora e acessível para usuários iniciantes.

As respostas devem ser apresentadas utilizando:

- listas organizadas;
- exemplos do cotidiano;
- linguagem simples;
- dicas práticas;
- passo a passo quando necessário.

Evite:

- linguagem excessivamente técnica;
- respostas genéricas;
- recomendações incompatíveis com a realidade financeira do usuário;
- informações sem explicação.

Ao final, apresente sugestões criativas para transformar essa ideia em um projeto Frontend interativo, indicando:

- estrutura das telas;
- componentes da interface;
- experiência do usuário (UX);
- recursos de Inteligência Artificial;
- possíveis gráficos;
- dashboards;
- notificações inteligentes;
- funcionalidades futuras que possam enriquecer o projeto.
